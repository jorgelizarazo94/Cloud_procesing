Introduction¶

Utilizando los datos recopilados a travs de redes de niebla entre los aos 2015 y 2019 en el Cerro Montezuma, rea adyacente al Parque Nacional Natural Tatam, llevamos a cabo un proceso de extraccin de informacin. Este proceso implic el anlisis de varios conjuntos de datos raster, incluyendo aquellos de WorldClim 30seg y Hansen GFC (2022 - treecover200), junto con su recorte mediante un archivo shapefile de tipo polgono con un buffer alrededor del rea de reserva (disponible en el enlace de GitHub proporcionado). Posteriormente, fusionamos dos bases de datos: una con informacin de campo y otra con datos extrados de los raster. Realizamos una limpieza exhaustiva de los datos para homogeneizar nombres de especies, eliminar informacin superflua y estandarizar formatos. Utilizando anlisis de componentes principales (PCA), determinamos el nmero mnimo de componentes necesario para representar el 90% de los datos. Luego, mediante el algoritmo de agrupamiento K-means, identificamos qu especies se agrupaban en qu comunidades, dividiendo as el conjunto en tres comunidades distintas. Finalmente, entrenamos un modelo de XGBoost para predecir a qu comunidad perteneceran las especies a partir de los datos extrados de los raster. Trasladamos este modelo a un escenario geogrfico y generamos distribuciones o regiones ideales para cada comunidad de aves en el rea del PNN Tatam y su zona circundante.

def objective(space):
    clf = xgb.train(space, dtrain, num_boost_round=1000, evals=[(dtest, "test")], early_stopping_rounds=30, verbose_eval=False)
    
    preds = clf.predict(dtest)
    # Asegrate de que 'preds' sean las clases predichas para calcular F1-score
    preds = np.asarray([np.argmax(line) for line in preds])
    f1 = f1_score(y_test, preds, average='weighted')  # 'weighted' considera el desbalance de clases
    
    # Hyperopt minimiza la funcin objetivo; se retorna 1 - F1 para maximizar el F1-score
    return {'loss': 1 - f1, 'status': STATUS_OK}

space = {
    'max_depth': hp.choice('max_depth', np.arange(1, 14, dtype=int)),
    'eta': hp.uniform('eta', 0.01, 0.3),
    'objective': 'multi:softmax',
    'num_class': 3,
    'lambda': hp.uniform('lambda', 1e-8, 1.0),  # Regularizacin L2
    'alpha': hp.uniform('alpha', 1e-8, 1.0)  # Regularizacin L1
}

trials = Trials()

    data = []
